# Aquarium

터미널 속 작은 힐링 어항. ASCII 열대어들이 헤엄치고, 공기방울이 올라오고, 시간이 지나면 아기 물고기가 태어납니다.

![어항 데모 — 물고기들이 헤엄치고, 먹이를 주면 몰려들고, 아기 물고기가 태어납니다](demo.gif)

## 설치

### 미리 빌드된 바이너리 (Swift 불필요, macOS 전용)

Apple Silicon과 Intel Mac을 모두 지원하는 유니버설 바이너리입니다:

```sh
curl -L https://github.com/agiletalk/Aquarium/releases/latest/download/aquarium.tar.gz | tar xz
./aquarium
```

### 소스에서 빌드 (Swift 5.9+)

```sh
swift run -c release
```

또는 빌드 후 바이너리 직접 실행:

```sh
swift build -c release
.build/release/aquarium
```

## 조작

| 키 | 동작 |
|----|------|
| `f` | 먹이 주기 — 물고기들이 먹이를 향해 몰려듭니다 |
| `q` / `Ctrl-C` | 종료 |

## 구경 포인트

- 물고기마다 색과 종이 다릅니다. 줄무늬 열대어(`><|||°>`)도 있어요.
- 공기방울은 수초와 물고기 입에서 올라오며, 수면에 가까워질수록 커집니다 (`.` → `o` → `O`).
- 3~5분마다 아기 물고기(`><>`)가 태어나고, 자라면 어른 물고기가 됩니다.
- 먹이를 자주 주면 번식이 빨라집니다. 최대 마릿수는 터미널 크기에 비례해요.
- 바닥의 보물상자 `[____]`가 이따금 덜컥 열리며 금화를 반짝이고 공기방울을 뿜습니다. 근처 물고기들은 깜짝 놀라 흩어져요.
- 해파리 `(___)`가 갓을 오므렸다 펴며 둥실둥실 떠다닙니다. 오므릴 때 위로 쏘옥, 펴면 천천히 가라앉아요.
- 터미널 크기를 조절하면 어항도 실시간으로 따라 커지고 작아집니다.
